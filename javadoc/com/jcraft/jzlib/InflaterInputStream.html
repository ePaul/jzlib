public class InflaterInputStream
extends java.io.FilterInputStream

An inflating stream wrapper around a compressed stream. This stream uses an Inflater to do the actual work - such an Inflater can either be passed to the constructor or will be created internally. (getInflater() can be used to access this Inflater.)
